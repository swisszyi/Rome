Create New Repository

1. Go to github.com/new

2. Set repository name to anything you like

3. Select Public visibility

4. Check Add a README file

5. Click Create repository

Launch GitHub Codespace

1. In your new repository, click the green <> Code button

2. Select the Codespaces tab

3. Click Create codespace on main

4. Wait for environment to load (approximately 20 seconds)

Verification & Troubleshooting

Verify Successful Commits

• Visit your repository on GitHub and click on the Commits tab

• Confirm 15 entries on June 10, 2025

Troubleshooting

Commits Not Appearing

Ensure you pushed your commits: git push origin main

Incorrect Timestamps

Verify Codespace timezone: date (should show UTC)

Git Configuration

Set your identity:
